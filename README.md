# Fake-News-Detection
we made this fake news detection app using nlp and machine learning and a flutter based mobile application which can help find a news headline real or fake.
in this project it use nlp technique to preprocess the news headline 
then extract with the help of TF-IDF method which converts the texts into nemerical vectors
then with the help of the machine learning model and the dataset it classify whether the news is real or fake
